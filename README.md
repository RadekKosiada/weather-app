# Weather Single Page Application

* displays temperature and sky condition/precipitation (as icon) on respective day every three hours;
* displays weekday, location and date for selected entry;

### Built with
* React.js;

### Available Scripts


* In the main directory, you can run:

#### `npm install` 

Installs all dependencies on your machine.

#### `npm run dev`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser. 

### IMPORTANT:

* To run the app, you also need an  `.env` file in project's root directory with the correct API Endpoint: e.g. API_ENDPOINT=https://your-endpoint.com

* If you start the script with the endpoint you will see an error message;
* Create an `.env` file with the correct endpoint and restart the project with `npm run dev`.
